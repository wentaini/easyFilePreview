使用方法

1. 完整预览模式

访问 /filePreview.html 进入完整文件预览页面

{{ baseUrl }}/filePreview.html?url=https://example.com/document.pdf

2. 简洁预览模式

访问 /filePreviewSimple.html 进入简洁预览页面（只显示文件内容）

{{ baseUrl }}/filePreviewSimple.html?url=https://example.com/document.pdf

3. iframe嵌入使用

将以下代码嵌入到您的页面中：

<!-- 完整预览模式 -->
<iframe src="{{ baseUrl }}/filePreview.html?url=YOUR_FILE_URL" width="100%" height="600"></iframe>

<!-- 简洁预览模式 -->
<iframe src="{{ baseUrl }}/filePreviewSimple.html?url=YOUR_FILE_URL" width="100%" height="600"></iframe>

4. 单元格复制功能

Excel和CSV文件支持单元格点击复制功能：

  • 点击表格中的任意单元格，内容自动复制到剪贴板
  • 支持现代浏览器的Clipboard API
  • 提供降级方案支持旧版浏览器
  • 显示复制成功提示信息
  • 鼠标悬停显示"点击复制"提示

💡 提示：在Excel和CSV预览页面中，将鼠标悬停在单元格上会显示提示，点击即可复制单元格内容。
